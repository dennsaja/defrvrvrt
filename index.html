Konfigurasi Supabase (data user) dan Google Sheets (data laporan)
Supabase menyimpan data akun teknisi (username, HP, password, tanggal daftar, dll).
1
Buka supabase.com → login → klik New Project
2
Isi nama project, password DB, pilih region Southeast Asia (Singapore)
3
Tunggu project siap, lalu buka Settings → API
4
Copy Project URL dan anon public key
5
Buka SQL Editor di sidebar, paste SQL di bawah, klik Run

Google Sheets menyimpan semua data laporan kegiatan. Riwayat di app diambil langsung dari spreadsheet.
1
Buat Google Sheets baru di spreadsheet.google.com
2
Klik Extensions → Apps Script
3
Hapus semua kode default, paste script di bawah, klik 💾 Save
4
Deploy → New deployment → Web App
Execute as: Me | Access: Anyone
5
Klik Authorize access → pilih akun Google → Allow
6
Copy URL deployment (diakhiri /exec) paste di bawah
⚠️ Pastikan pilih Anyone (bukan "Anyone with Google Account") agar bisa terima request dari app.
